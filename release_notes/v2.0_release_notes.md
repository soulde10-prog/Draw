# Release Notes for v2.0

## Release Date:
2026-04-13

## Comprehensive Documentation of 3D Visualization Features
- New rendering engine with photorealistic graphics
- Enhanced user interface for easier navigation and manipulation of 3D models
- Support for real-time shading and shadowing techniques
- Improved lighting options including dynamic and ambient lighting

## Technical Specifications
- **Framework:** Three.js v0.129.0
- **WebGL Version:** 2.0
- **Supported Formats:** OBJ, FBX, GLTF
- **Performance Improvements:** Optimized algorithms for better rendering speed and reduced memory usage

## API Changes
- Added new methods for handling 3D model transformations:
  - `translateModel(x, y, z)` - Translates the model by the specified vector.
  - `rotateModel(axis, angle)` - Rotates the model around a given axis.
  - `scaleModel(scalingFactor)` - Scales the model by a specified factor.
- Deprecated methods:
  - `oldTranslate()` - Use `translateModel()` instead.

## Migration Guide
1. Update your project to include the new rendering engine library.
2. Replace deprecated methods with their new counterparts in your codebase.
3. Test your models to ensure compatibility with the new API changes.
